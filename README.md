# 2D Brick Breaker Game (SFML)
A classic 2D Brick Breaker game developed in C++ using the SFML graphics library, featuring real-time collision physics, dynamic scoring, and smooth frame-rate optimization.
An interactive, high-performance **2D Brick Breaker** game built from scratch using **C++** and the **SFML (Simple and Fast Multimedia Library)** framework. The project demonstrates core object-oriented programming (OOP) principles, real-time rendering, and collision handling techniques.

## 🎮 Game Features & Mechanics

* **Fluid Controls:** Smooth paddle movement synchronized with real-time user keyboard inputs.
* **Physics & Collisions:** Real-time bounce mechanics calculating ball reflections off walls, the paddle, and individual bricks.
* **Dynamic Grid:** A structured layout of breakable bricks that track state and disappear upon collision.
* **On-Screen UI:** Real-time score display and game state tracking rendered using SFML's Text system.
* **Performance Optimized:** Configured with a stable framerate limit for a smooth and consistent runtime environment across different hardware.

## 🛠️ Tech Stack & Concepts Used

* **Language:** C++
* **Graphics Library:** SFML 2.6.x (Graphics, Window, System)
* **Core Concepts:** Object-Oriented Programming (Classes for Ball, Brick, and Paddle), Vector Mathematics, Game Loop Architecture.
